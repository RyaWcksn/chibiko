**Endpoints**

There's serveral endpoint interface that user can interact with, they are :

|----------+-----------------------------------------|
| Url      | Decscription                            |
|----------+-----------------------------------------|
| /endcode | Endcode the url given in body payload   |
| /{code}  | Redirecting code to the actual endpoint |
